React Native for Androidを紹介.

<!--more-->

```sh
brew install android-sdk
```

```sh
export ANDROID_HOME=/usr/local/opt/android-sdk
```

を`~/.bashrc`又は`~/.zshrc`に追加し再読み込み.

```sh
source ~/.zshrc
# source ~/.bashrc
```
